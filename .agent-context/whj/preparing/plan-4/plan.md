# 壁纸、搜索栏与设置

> 状态: 未执行

## 目标

实现每日壁纸展示与刷新、搜索栏（必应/Google 引擎切换）、完整设置面板（搜索栏可见性、壁纸提供商、自动刷新、**网站圆角大小**、备份/导入）。完成后项目具备完整的新标签页体验。

## 内容

### 1. 壁纸服务

- 壁纸提供商抽象层（`WallpaperProvider` 接口），支持扩展
- 内置提供商：Bing 每日壁纸（`https://www.bing.com/HPImageArchive.aspx`）、Unsplash Random
- 壁纸缓存：下载图片存入 IndexedDB（Blob），避免重复网络请求
- 刷新逻辑：手动刷新（按钮）+ 自动刷新（可配置间隔，默认每日）
- 完成标准：可从 Bing/Unsplash 获取壁纸，缓存命中时无网络请求

### 2. 壁纸展示

- 遵循 `ui-ux-pro-max` 设计系统
- 全屏背景层：`position: fixed`、`object-fit: cover`
- 渐进式加载：先显示低分辨率模糊缩略图（CSS `filter: blur`），高清图加载完成后渐显切换（`opacity` 过渡）
- 加载失败 fallback：优雅的纯色/渐变背景
- 刷新按钮：页面右下角浮动，旋转加载动画
- 壁纸之上添加半透明遮罩层确保内容可读性
- 完成标准：壁纸全屏展示、切换流畅无闪烁、失败时 fallback 优雅

### 3. 搜索栏

- 遵循 `ui-ux-pro-max` 设计系统
- 位置：页面上方居中
- 外观：毛玻璃/半透明背景、圆角、柔和阴影
- 输入框：聚焦时宽度展开动画、placeholder 引导文案
- 搜索引擎切换：输入框左侧图标按钮，点击切换（Bing ↔ Google），显示当前引擎 logo
- 回车触发：用当前引擎在当前标签页打开搜索结果
- 搜索引擎配置存入 localStorage
- 完成标准：可输入关键词、切换引擎、回车搜索，动画流畅

### 4. 设置面板

- 入口：右上角齿轮图标按钮
- 面板形式：右侧滑入抽屉（slide-in drawer）
- **动态 import** 按需加载设置面板组件，不影响首屏体积
- 遵循 `ui-ux-pro-max` 设计系统
- 完成标准：设置面板按需加载，打开/关闭动画流畅

### 5. 设置项实现

- **搜索栏**：显示/隐藏开关（Toggle）
- **壁纸提供商**：下拉选择（Bing / Unsplash）
- **壁纸自动刷新**：开关
- **壁纸刷新间隔**：数值输入或预设选项（1小时 / 6小时 / 每日 / 每周）
- **网站圆角大小**：
  - 范围滑块（`<input type="range">`），最小值 `0`（直角），最大值 `50`（近似圆形），步长 `1`
  - 滑块两端显示最小/最大圆角预览图示
  - 实时预览：拖动时网格中的网站卡片圆角即时更新（通过 CSS 变量 `--card-radius` 驱动）
  - 默认值 `12`（对应 `rounded-xl`），存入 localStorage
- 所有设置实时存入 localStorage，修改即生效（无需保存按钮）
- 完成标准：所有设置项可操作，修改后立即生效，刷新页面设置保持；圆角滑块实时预览效果流畅

### 6. 备份与导入

- **导出**：将全部数据（网站列表、分组、设置）序列化为 JSON，触发浏览器下载
- **导入**：选择 JSON 文件，解析并校验数据格式与完整性
  - 校验失败：显示错误提示，不覆盖现有数据
  - 校验通过：确认覆盖 → 写入 IndexedDB + localStorage → 刷新页面状态
- UI：设置面板底部的导出/导入按钮
- 完成标准：导出文件可被导入恢复，数据完整性校验覆盖异常格式

### 7. UI 精打磨与最终验收

- 全局交互动画审查（过渡、微动效）
- 键盘快捷键：`/` 聚焦搜索栏、`Esc` 关闭弹窗/设置
- 按 `ui-ux-pro-max` Pre-Delivery Checklist 逐项验证
- 首屏性能验收：Lighthouse 或手动计时，确认 < 200ms 目标
- `bun run build` 产物体积审查，确认 JS < 50KB gzip
- 完成标准：通过 Pre-Delivery Checklist 全部项目，性能达标
