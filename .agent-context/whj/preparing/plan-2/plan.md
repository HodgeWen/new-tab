# 网站管理与网格布局

> 状态: 未执行

## 目标

实现核心网站管理功能：网站 CRUD、网格布局展示（图标 + 名称，1×1 单元格）、拖拽排序、批量选择与批量操作、自定义图标上传（含裁切）、数据持久化。完成后可在新标签页中添加/删除/编辑网站，以网格形式展示并支持拖拽排序和批量操作。

## 内容

### 1. 网站数据模型与状态管理

- 定义 `Website` 类型：`id`、`name`、`url`、`icon`（Google Favicon URL | base64 自定义图标）、`order`、`groupId?`
- 创建 `websitesStore`（Preact Signals）：网站列表 signal、排序 computed、选中集合 signal（`Set<string>`）
- 实现 CRUD 方法：`addWebsite`、`updateWebsite`、`removeWebsite`、`reorderWebsite`
- 实现选择方法：`toggleSelect`、`selectAll`、`clearSelection`
- 启动时从 IndexedDB 加载数据到 signal，同时用 localStorage 缓存首屏数据以加速
- 所有变更实时写入 IndexedDB，防抖批量写入 localStorage 缓存
- 完成标准：store 方法可独立调用，数据持久化正确

### 2. 网站卡片组件

- 遵循 `ui-ux-pro-max` 设计系统
- 正方形卡片占 1×1 网格单元
- 图标展示优先级：
  1. 用户上传的自定义图标（存储为 base64，取自 IndexedDB）
  2. Google Favicon API（`https://www.google.com/s2/favicons?domain=...&sz=64`）
  3. 失败时 fallback 到首字母头像（取域名首字母，随机背景色）
- 网站名称：单行截断，居中显示于图标下方
- 交互状态：hover 高亮、active 缩放反馈
- **批量选择模式**：进入选择模式后卡片左上角显示复选框；已选中状态显示勾选标记和高亮边框
- 点击打开网站（当前标签页 `location.href` 或 `window.open`）
- 完成标准：卡片渲染正确，图标加载含 fallback，批量选择视觉反馈清晰，点击可打开网站

### 3. 自定义图标上传与裁切

- **上传入口**：编辑网站弹窗中提供「上传自定义图标」按钮（`<input type="file" accept="image/*">`）
- **裁切 UI**：
  - 上传后弹出裁切对话框（Modal）
  - 使用 `<canvas>` 实现裁切预览，不引入第三方裁切库
  - 支持拖拽移动裁切框，固定比例 1:1（正方形）
  - 提供缩放滑块调整显示比例
  - 「确认」按钮将裁切区域绘制到 canvas 并导出为 base64（`image/webp`，质量 0.8）
- **存储**：将 base64 字符串作为 `icon` 字段存入 IndexedDB（不用 localStorage，体积较大）
- **清除**：编辑弹窗中提供「使用默认图标」选项，清空自定义图标恢复 favicon
- 完成标准：上传图片后可裁切为正方形，保存后卡片展示自定义图标；刷新后图标持久化

### 4. 网格布局

- CSS Grid 实现，列数响应式（`repeat(auto-fill, minmax(90px, 1fr))` 或固定列数）
- 网格间距统一（gap），与分组尺寸兼容
- 空位视觉处理
- 完成标准：网站以网格排列，窗口缩放时布局自适应

### 5. 添加网站

- 添加按钮：`+` 图标卡片，样式与网站卡片统一，始终排在最后
- 点击弹出添加表单（Modal 或 Popover）
- 表单字段：URL（必填）、名称（选填，可自动获取）、自定义图标（选填）
- URL 自动补全 `https://` 协议头
- 提交后自动获取 favicon（若无自定义图标）
- 完成标准：可通过 UI 添加网站并立即在网格中显示

### 6. 编辑与删除

- 右键上下文菜单：编辑、删除、移动到分组（单个）、选择多个
- 编辑复用添加表单，预填数据（含自定义图标上传/裁切入口）
- 删除需确认（轻量确认，非系统 confirm）
- 「选择多个」触发批量选择模式
- 完成标准：右键菜单可触发编辑/删除/选择，操作后网格实时更新

### 7. 批量选择与批量操作

- **进入选择模式**：
  - 右键网站卡片 → 选择「选择多个」
  - 或长按（移动端兼容，`500ms` touchstart）
- **选择模式 UI**：
  - 顶部或底部出现操作工具栏（浮动 bar）
  - 工具栏显示：已选数量、「全选」、「取消」、「移动到分组...」、「删除」
  - 每张网站卡片左上角显示复选框
  - 点击卡片切换选中/取消状态（不打开网站）
- **批量移动到分组**：
  - 工具栏点击「移动到分组...」→ 弹出分组列表 Popover
  - 选择目标分组后，将所有选中网站加入该分组，清空选择状态
  - 若目标分组容量不足，超出部分留在主网格并提示
- **批量删除**：轻量确认后批量移除
- **退出选择模式**：点击「取消」或按 `Esc`
- 完成标准：可进入批量选择模式，选中多个网站后可批量移动到分组或批量删除

### 8. 拖拽排序

- 使用原生 HTML Drag & Drop API（零依赖）
- 拖拽开始：源卡片半透明
- 拖拽中：目标位置显示占位指示器
- 拖拽结束：更新 order 并持久化
- 批量选择模式下禁用拖拽（避免交互冲突）
- 触摸设备适配（`touch-action` 处理）
- 完成标准：可拖拽网站卡片重新排序，松手后顺序持久化
